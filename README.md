## ?? Project Goal 
The aim of this project is to analyze and predict average temperature in the agri-food sector, using data from FAO and IPCC. The goal is to understand climate impacts and develop sustainable strategies for stakeholders, including policymakers and agricultural businesses. 
 
## ?? Dataset 
- Features: 30+ 
- Records: ~7000 
- Sources: FAO, IPCC 
- Coverage: Global agri-food sector
